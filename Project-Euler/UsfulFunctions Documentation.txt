IsPrime(long num)						 Checks if num is a prime number.

IsPalindrom(long num)				   	 Checks if num is a palindrom (reads the same both ways).

ReverseString(string str)				 Etention method for string class.
										 Returns str from end to beginning.

GetPrimeFactors(long num)				 Gets all the prime factors of num.

SumOfSquersSequence(int limit)			 Returns the sum of 1^2 + 2^2 + 3^2 +...+ limit^2.

SumOfArithmeticSequence(int beginning,
						int limit,
						int difference)  Returns the sum of an arithmetic sequence.

Divide(ref long num, long divider)	     Divides num by divider as many times as possible.
